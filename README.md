# MHYSA

## Installation

1. Clone this repo
2. `cd mhysa`
3. `composer install`
4. `npm install`
5. `npm run dev`
6. `cp .env.example .env`
7. `php artisan key:generate`
8. Update database configuration
9. `php artisan migrate:fresh --seed`
10. `php artisan serve`
